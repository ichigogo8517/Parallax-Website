視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網站設計技巧，讓網站的背景移動速度比前景慢。這種處理會在訪客向下捲動頁面時產生 3D 效果，增添深度並打造更具沉浸感的瀏覽體驗。視差效果建立在視覺錯覺上。由於人眼會把靠近的物體看得比遠處物體大，因此我們會覺得遠處的物體移動得比較慢。這種錯覺長久以來被運用在各種媒介的視差處理中，營造出逼真的效果。最早的案例可追溯到傳統動畫，例如迪士尼的《白雪公主》，以及《超級瑪利歐》等電玩。隨著 CSS 與 HTML 的進步，視差效果也逐漸演變成現今常見的網頁設計手法。

2007 年，網頁開發者 Glutnix 在部落格文章中分享了最早的瀏覽器視差滾動實作之一，提供了支援當時 IE6 等瀏覽器的 JavaScript 與 CSS2 範例程式與示範。不過直到 2011 年 HTML5 與 CSS3 問世後，這項技術才在網頁設計師之間流行起來。擁護者認為，這是擁抱網頁流動性的一種簡單方式。他們利用視差背景提升與使用者的互動，改善整體的網站體驗。然而，普渡大學在 2013 年發表的研究指出：「……雖然視差滾動強化了使用者體驗的某些面向，但並不一定能提升整體的使用者體驗。」


為什麼我該使用視差圖像？

    設計師應該嘗試視差圖像有兩個主要原因：

  • 有助於說故事

    視差圖像非常適合用來說故事。好的設計就是溝通，而在設計網站時，我們希望能與訪客互動，講述關於產品、服務或品牌的精彩故事。視差滾動能讓網站訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓訪客不用切換頁面就能讀完整個品牌故事。

  • 有助於提升使用者參與度

    視差圖像能提升使用者參與度。降低網站跳出率是設計時的重要目標。跳出率衡量的是訪客在瀏覽單一頁面後就按下返回或關閉分頁的比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的作法都值得嘗試。精心設計的視差頁面能提供動態體驗，讓使用者在頁面停留更久。


如果想在網站加入視差滾動效果，可以參考以下建議：

  • 選擇性使用

    當你只在特定元素上使用視差效果，而不是所有頁面都套用時，效果會更強烈。可將它用在標題、首圖，或只在首頁呈現。視差滾動的目的是強化網站設計，而不是分散使用者注意力。

  • 注意影像壓縮

    視差滾動需要使用各種媒體檔案與 CSS 形狀來營造深度感。請避免使用過大的影像檔案，以免載入太慢而影響網站效能與使用體驗。

  • 善用色彩

    視差滾動不只關乎前景與背景圖像，也與色彩有關。你可以在背景使用較低飽和度的色調，在前景使用更鮮明的顏色，打造更強烈的層次感。
